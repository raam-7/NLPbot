Types of Ambiguity
Lexical ambiguity
A word that could be used as a verb, noun, or adjective.
Semantic ambiguity
The interpretation of a sentence in context. E.g. “I saw the boy on the beach
with my binoculars” — I saw the boy through my binoculars, or the boy had
my binoculars with him.
Syntactic ambiguity
Structure creates the confusion of meaning — “with my binoculars” could
modify the verb “saw” or the noun “boy”.
Even for humans this sentence is difficult to interpret without surrounding context. POS (part-of-speech) tagging is one NLP solution
that can help solve the problem, somewhat.

Text Preprocessing — Steps & Python Code
1. Lowercasing
"NLP" → "nlp"
2. Remove Punctuation
"Hello, world!" → "Hello world"
3. Remove Stop Words
"the cat sat on mat" → "cat sat mat"
4. Remove HTML/Noise
"<p>Buy now!</p>" → "Buy now"
5. Normalize Numbers
"I have 3 cats" → "I have NUM cats"
python — NLTK preprocessing pipeline
import re, nltk
from nltk.corpus import stopwords
from nltk.tokenize import word_tokenize
def preprocess(text):
text = text.lower()
text = re.sub(r'[^\w\s]', '', text)
text = re.sub(r'\d+', 'NUM', text)
stop = set(stopwords.words('english'))
tokens = word_tokenize(text)
return [w for w in tokens
if w not in stop]
# Test
preprocess("The Cat SAT on 3 Mats!!")
# ['cat', 'sat', 'NUM', 'mat']

python — normalization with regex & contractions
import re
import unicodedata
CONTRACTIONS = {
"don't": "do not",
"i'm" :
"i am",
"won't": "will not"}
def normalize(text):
text = text.lower()
for k,v in CONTRACTIONS.items():
text = text.replace(k, v)
text = re.sub(r'https?://\S+','URL', text)
return text
normalize("I don't visit https://sit.edu")
# → "i do not visit URL"

===== PAGE 92 =====

Classroom Explanation: 2 Minutes
A ready-to-speak explanation for students.
"Imagine every word as a point on a map. In One-Hot Encoding, every word is isolated, so
the computer knows only that they are different. Word2Vec changes this by placing
similar words close together - 'king' near 'queen', 'doctor' near 'hospital', and 'cat' near
'dog'. BERT goes a step further: it understands that the same word can have different
meanings depending on the sentence. For example, 'bank' in 'river bank' and 'bank' in
'bank account' are represented differently. This evolution - from One-Hot to Word2Vec to
BERT - is one of the biggest milestones in modern NLP."
This connects the concept with intuitive examples and practical code students can relate to.

Chunking
• Chunking involves grouping together individual pieces of information
from a sentence, such as nouns, verbs, adjectives, and adverbs, into
larger units known as chunks.
• The most common type of chunking is noun phrase (NP) chunking, which
involves identifying and extracting noun phrases from a sentence, such
as "the cat," "a book," or "my friend." Another type of chunking is verb
phrase (VP) chunking, which involves identifying and extracting verb
phrases from a sentence, such as "ate breakfast," "is running," or "will
sing.“
• Steps:
• Identify the parts of speech in a sentence using part-of-speech (POS)
tagging.
• The chunks are created by grouping together certain types of words
based on their POS tags.
• The most common types of chunks include
• noun phrases
• verb phrases,
• prepositional phrases.

===== PAGE 118 =====

Chinking
• Chinking, on the other hand, is the process of excluding certain words or
phrases from a chunk. This is useful when we want to exclude specific
types of words, such as prepositions, conjunctions, or determiners, from
the chunks we extract.
• Chinking is typically used in combination with chunking, and it involves
identifying the words or phrases that we want to exclude from a chunk
and marking them with the tag O (outside) in a named entity recognition
(NER) system.

"I love NLP!" →

→ ['i','love','nlp','!']

python — end-to-end NLP pipeline
import nltk, spacy, re
from nltk.corpus import stopwords
from nltk.stem import WordNetLemmatizer
def nlp_pipeline(text):
# Step 1-4: clean & tokenize
text = text.lower()
text = re.sub(r'[^\w\s]','',text)
stop = set(stopwords.words('english'))
tokens = [w for w in text.split()
if w not in stop]
# Step 5: lemmatize
lem = WordNetLemmatizer()
tokens = [lem.lemmatize(w) for w in tokens]
# Step 6-7: POS + NER via spaCy
nlp = spacy.load('en_core_web_sm')
doc = nlp(' '.join(tokens))
return doc
